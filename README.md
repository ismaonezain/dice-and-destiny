# Dice and Destiny

## Game Description
Dice and Destiny is an immersive tabletop game that combines elements of chance and strategy, allowing players to embark on adventures, face challenges, and gain rewards through dice rolls and decision-making.

## Game Modes
### Free Mode
In Free Mode, players can enjoy the basic features of the game without any cost. This mode is ideal for new players looking to understand the game mechanics.

### Paid Mode
Paid Mode offers additional features, enhanced gameplay elements, and exclusive rewards. Players can purchase access to this mode for a premium experience.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/<owner>/dice-and-destiny.git
   cd dice-and-destiny
   ```
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Run the game:
   ```bash
   npm start
   ```  

## Smart Contract Details
The game utilizes smart contracts to manage transactions and game states securely.

## Key Features
- **Interactive Gameplay**: Engaging mechanics that involve dice rolls and player decisions.
- **Multiple Game Modes**: Options for both free and premium players.
- **Smart Contract Integration**: Ensures transparency and security for in-game purchases.

## Contract Functions
- `rollDice()`: Simulates a dice roll and returns the result.
- `purchaseAccess()`: Allows players to purchase premium access in Paid Mode.
- `getPlayerStats()`: Fetches the stats of a player.
